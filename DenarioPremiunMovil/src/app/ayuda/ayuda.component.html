Pedido

Este módulo está diseñado para la gestión de pedidos en linea. Mediante el dispositivo los ejecutivos de ventas pueden tomar los pedidos desde el mismo local de su cliente o también al llegar su departamento de ventas en tan sólo unos minutos.

Crear Nuevo Pedido

Ingresar al módulo Pedidos seleccionando en el menú principal del Sistema.

Pulsar la opción Nuevo Pedido. El sistema mostrará una pantalla titulada Nuevo Pedido y se apreciarán tres (3) pestañas diferentes: General, Pedido, Total.

En la pestaña General el usuario debe completar los campos solicitados a continuación, para ello deberá pulsar sobre cada campo de texto para elegir los valores: Cliente, Condición de pago, Nº orden de compra, Persona responsable del pedido, Solicitud de despacho: Por Aprobar.

En la pestaña Pedido se debe ingresar el pedido final de los productos, para ello se debe efectuar la búsqueda del producto y luego pulsar sobre el producto seleccionado e indicar la cantidad del producto en la unidad disponible.

Si el producto se ofrece en varias unidades de venta al seleccionar un producto pulse sobre el botón desplegable de la unidad para elegir la unidad en la que desee pedir el producto.

Si se manejan varias listas de precio para un producto al seleccionar un producto pulse sobre el botón desplegable de la lista para elegir la el precio en la que desee pedir el producto.

La pestaña Total muestra el monto final del pedido y las cantidades pedidas por cada unidad de productos si aplica, podrá visualizar el detalle de cada ítem pedido pulsando sobre el producto que desee explorar.

Luego de llenar la información el usuario debe pulsar el botón Guardar ubicado en la esquina superior derecha y luego pulsar el botón Enviar para registrarlo y enviarlo al sistema central.

Modificar Pedido

Para modificar un pedido el mismo debe tener estatus Guardado, si el pedido fue enviado al sistema central ya no podrá modificarse.

Ingresar al módulo Pedidos seleccionando en el menú principal del Sistema.

Pulsar la opción Buscar Pedido.

El sistema mostrará una pantalla titulada Buscar Pedido y la lista de los pedidos que han sido cargados.

Pulse sobre el pedido que desea modificar, automáticamente el sistema mostrará la pantalla del pedido seleccionado con las tres (3) pestañas: General, Pedido y Total.

En la pestaña General podrá editar los valores que requiera, para ello pulse sobre cuadro de texto y efectúe la nueva selección para el campo correspondiente.

En La pestaña Pedido se pueden editar o modificar el pedido, agregando o eliminado productos. Si requiere agregar nuevos productos, debe efectuar la búsqueda del producto y luego pulsar sobre el producto seleccionado e indicar la cantidad del producto en la unidad disponible. Si requiere eliminar productos, debe efectuar la búsqueda del producto, luego pulsar sobre el producto seleccionado y borrar el valor colocado en la cantidad pedida.

La pestaña Total muestra el monto final del pedido y las cantidades pedidas por cada unidad de productos si aplica, podrá visualizar el detalle de cada ítem pedido pulsando sobre el producto que desee explorar.

Luego de llenar la información el usuario debe pulsar el botón Guardar ubicado en la esquina superior derecha y luego pulsar el botón Enviar para registrarlo y enviarlo al sistema central.

 Consultar Pedidos

Ingresar al módulo Pedidos seleccionando en el menú principal del Sistema.

Pulsar la opción Buscar Pedido.

El sistema mostrará una pantalla titulada Buscar Pedido y la lista de los pedidos que han sido cargados.

Pulse sobre el pedido con estatus enviado que desea consultar, automáticamente el sistema mostrará la pantalla del pedido seleccionado con las tres (2) pestañas: General y Total. No se podrá modificar campos con el estatus enviado.

Eliminar Pedido

Ingresar al módulo Pedidos seleccionando en el menú principal del Sistema.

Pulsar la opción Buscar Pedido.

El sistema mostrará una pantalla titulada Buscar Pedido y la lista de los pedidos que han sido cargados.

Pulse sobre la “X” ubicada al final del reglón del pedido que desea eliminar en el dispositivo.

El sistema le solicitará una confirmación para eliminar el pedido, una vez confirmada la acción se actualizará la lista desapareciendo de ésta el pedido eliminado.
Clientes

Seleccionar los parámetros para filtrar la búsqueda del cliente por: Nombre, Código o Zona.

Pulse sobre el cliente deseado para consultar el detalle del cliente.

El sistema mostrará dos pestañas: Cliente y Documentos de Venta en las cuales sólo puede realizar consultas de la información.

Para consultar documentos de ventas se debe pulsar sobre esta pestaña, el sistema desplegará los documentos.

Para visualizar el detalle de un documento de venta, pulse sobre el botón Ver, automáticamente el sistema mostrará los siguientes campos: Tipo de documento, Código del documento, Fecha del documento, Fecha de Vencimiento, monto, saldo, entre otros datos.

 Productos

Este módulo está diseñado para consultar los datos básicos de los productos en línea. Mediante el dispositivo los ejecutivos de ventas pueden consultar el nombre, las unidades de venta y la presentación de los productos. También permite ver el precio del producto, en las distintas listas en que se encuentre disponible, entre otros datos.

1. Al ingresar, el sistema le permitirá efectuar búsquedas de productos, seleccionando una letra del panel izquierdo, se mostrará una lista de los productos cuyos nombres comiencen con la letra seleccionada.

2. También puede realizar exploraciones rápidas partiendo de la lista de productos mostrada, escribiendo las primeras letras del código o del nombre del producto en la opción: Búsqueda…

3. Para visualizar el detalle de un producto deberá:
3.1. Pulsar sobre el producto que desea consultar.
3.2. El sistema mostrará una nueva pantalla con la ficha del detalle indicando: Nombre, Código, Línea Producto, Unidad de Venta, Cantidad en Inventario (si aplica) y el precio según lista seleccionada.

4. Si se manejan varias listas de precio para un producto podrá visualizar los distintos precios pulsando sobre el menú desplegable de la Lista y automáticamente el sistema cargará el precio correspondiente a la lista seleccionada.
