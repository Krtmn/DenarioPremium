# Project Brief

## Nombre
Denario Premium Móvil

## Propósito
Aplicación móvil para operación comercial en campo con enfoque offline-first, permitiendo ejecutar procesos de ventas y cobranza con sincronización posterior contra backend.

## Objetivos principales
- Permitir operación continua con conectividad intermitente.
- Reducir tiempos de gestión comercial en campo.
- Mantener trazabilidad y consistencia de transacciones.

## Usuarios objetivo
- Vendedor
- Transportista
- Promotor
- Cliente (perfil restringido)
- Soporte (según configuración)

## Alcance funcional principal
- Clientes
- Productos
- Pedidos
- Cobros
- Devoluciones
- Inventarios
- Depósitos
- Visitas/Despachos
- Sincronización

## Restricciones clave
- Dependencia de SQLite nativo en móvil (Android/iOS).
- El comportamiento web no replica completamente el entorno móvil.
